# Security Policy

## Supported Versions

Older versions of Tinkerme Share do not receive security updates. To ensure your system remains secure, we strongly recommend updating regularly. You can automate these updates using tools like [Watchtower](https://github.com/containrrr/watchtower).

## Reporting a Vulnerability

Thank you for taking the time to report a vulnerability. Please DO NOT create a public issue on GitHub because the vulnerability could get exploited.

Instead, please open a [private security advisory](https://github.com/tinkermesomething/tinkerme-share/security/advisories/new) on GitHub.
