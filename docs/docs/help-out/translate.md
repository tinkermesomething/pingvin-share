# Translating

You can help to translate Tinkerme Share into your language.

Translation files are located in `frontend/src/i18n/translations/`. To add or update a translation:

1. Fork the repository
2. Copy an existing translation file (e.g., `en-US.ts`) as a template
3. Translate the strings
4. Submit a pull request

Want to add a new language? Feel free to [open an issue](https://github.com/tinkermesomething/tinkerme-share/issues/new) or submit a PR directly.
